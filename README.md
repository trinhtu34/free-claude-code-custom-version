Project này được clone từ Project Free Claude Code về

Tôi có custom thêm, để có thể sử dụng thêm nhiều model khác, để có nhiều lựa chọn hơn cho các Dev. 

Hiện tại tôi đã làm thêm để có thể sử dụng Qwen, cả ở local, cả việc dùng API

Hiện tại hỗ trợ các Model, tool như sau

- Qwen ( cả local và API )
- Deepseek
- LM Studio
- LLAMA
- NVIDIA NIM API
- OPEN ROUTER

Nếu bạn muốn sử dụng Qwen ở local thì ở .env đặt QWEN_USE_API_KEY=false, nếu sử dụng API key thì QWEN_USE_API_KEY=true.
